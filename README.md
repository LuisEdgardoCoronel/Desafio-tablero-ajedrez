# Desafio-tablero-ajedrez
Te desafiamos a desentrañar un misterio: imagina a un chico disfrutando mientras colorea su tablero de ajedrez. Planea llenar cada cuadro con matices rojos o azules. Con el objetivo de añadir un toque único, desea mantener un balance entre los espacios rojos y azules, evitando al mismo tiempo que dos filas o columnas tengan la misma cantidad de espacios rojos.
